# TetrisProject01
<h2>テトリスを作るプロジェクト第一弾!!</h2>

久しぶりにC++とDxLibでゲームを作ってみるプロジェクトです。<br>
まだちゃちなタイトル画面を出して、キーボードでAを押すと閉じる機能しかありません。<br>
これからつけるつもりの機能は以下の通りです。<br>
<hr>
<ol>
<li>一人でキーボードを用いてテトリスを遊べるようにする。</li>
<li>二人でキーボードを用いてテトリスで通信対戦できるようにする。</li>
<li>ゲームパッドに対応する。</li>
<li>キーコンフィグに対応する。</li>
<li>音を出す。</li>
</ol>
